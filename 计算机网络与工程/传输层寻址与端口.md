传输层使用端口 (Port) 来标识主机中的应用进程，从而实现复用和分用。

- 端口号：16 位二进制，范围 $0 \sim 65535$。
- 分类：
    - 熟知端口 (Well-known Ports)：$0 \sim 1023$，由 IANA 指派给特定协议（如 HTTP 80, FTP 21）。
    - 登记端口 (Registered Ports)：$1024 \sim 49151$，为没有熟知端口的应用使用，必须登记。
    - 客户端口 (Ephemeral Ports)：$49152 \sim 65535$，留给客户进程暂时使用。

## 套接字 (Socket)

在因特网中，套接字唯一标识了网络中的一个进程：

$$\text{Socket} = (\text{IP address}, \text{Port number})$$

- UDP 分解：仅使用（目的 IP, 目的端口）二元组标识套接字。
- TCP 分解：使用（源 IP, 源端口, 目的 IP, 目的端口）四元组标识套接字。
