- 基于 [[TCP协议]]，端口号为 **25**。
- **推 (Push) 协议**：将邮件从发送方服务器推送到接收方服务器。
- 报文必须以 7 位 ASCII 格式。
- 使用持久连接。

### SMTP 与 HTTP 的比较 #重点

| 特性 | SMTP | HTTP |
| --- | --- | --- |
| 传输模式 | **推 (Push)**：发送方主动发起连接 | **拉 (Pull)**：接收方主动请求数据 |
| 报文格式 | 必须为 7 位 ASCII 格式 | 无限制 |
| 连接方式 | 持久连接 | 非持久或持久连接 |
| 交互方式 | 均为 ASCII 命令/响应交互，使用状态码 | 均为 ASCII 命令/响应交互，使用状态码 |
