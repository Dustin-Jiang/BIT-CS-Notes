衡量计算机性能需要从多个维度进行评估，这些指标在 408 考试中经常作为考点。

## CPU 频率指标

- **主频 (Main Frequency)**: CPU 内部数字脉冲信号震荡的速度。
  - **公式**: $$主频 = 外频 \times 倍频系数$$ #重点
- **外频 (External Frequency)**: 主板提供的基准频率，决定了 CPU 与外界（内存、外设）通讯的同步速度。
- **倍频系数**: CPU 主频与外频之间的比例。
- **超频技术**: 通过降低外频、提高倍频系数，使主频超过额定工作频率。 #重点

## 核心指标

1. **机器字长 (Word Length)**:
    - CPU 一次能处理的二进制位数（如 32 位、64 位）。
    - 由加法器、寄存器、总线位数决定。
2. **存储容量**:
    - 主存容量：通常以字节 (Byte) 为单位。
    - 寻址空间：由地址总线位数决定（如 32 位地址总线理论对应 4GB 寻址空间）。 #重点
3. **运算速度**:
    - **CPI** (Cycles Per Instruction): 执行一条指令所需的时钟周期数。
    - **MIPS** (Million Instructions Per Second): 每秒执行百万条指令数。 #重点
        - $MIPS = \frac{指令条数}{执行时间 \times 10^6} = \frac{主频}{CPI}$
    - **MFLOPS** (Million Floating-point Operations Per Second): 每秒执行百万次浮点运算数。 #重点
    - **对比**: MIPS 衡量整数运算，MFLOPS 衡量浮点运算。在 408 考试中，这两个概念常以选择题形式出现。
4. **吞吐量 (Throughput)**: 系统在单位时间内处理请求的数量。
5. **响应时间 (Response Time)**: 从提交请求到系统产生响应的时间。

## 常用单位与换算基础

教师要求必须熟练掌握 $2^0$ 到 $2^{10}$ 的数值，以及大单位换算： #重点
- $2^{10} = 1024 = 1K$
- $2^{20} = 1M$
- $2^{30} = 1G$
- $2^{40} = 1T$
- **字长 vs 字节**:
    - **位 (Bit)**: 描述 0/1 状态的最小单位。
    - **字节 (Byte)**: 计算机表示有效信息的**基本单位**，$1 Byte = 8 Bits$。
    - **字 (Word)**: CPU 一次处理的信息量，其长度（字长）随 CPU 架构不同而异。

## 32 位系统与 4GB 寻址空间
- **理论限制**: $2^{32} = 4GB$。
- **实际情况**: 32 位操作系统（如 Win7）由于系统预留及 I/O 映射，实际可用空间仅 3.1GB~3.2GB 左右。 #重点
- **解决方案**: 若内存大于 4GB，必须安装 64 位操作系统才能充分利用。
