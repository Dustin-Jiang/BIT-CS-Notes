主存储器主要由半导体随机存取存储器 (RAM) 组成，根据记忆单元的物理性质分为静态 RAM 和动态 RAM。

## 对比

| 特性 | SRAM (静态 RAM) | DRAM (动态 RAM) |
| :--- | :--- | :--- |
| **存储原理** | 双稳态触发器 | 栅极电容 (电荷) |
| **是否需要刷新** | 否 | 是 (需 [[DRAM刷新]]) |
| **速度** | 快 | 慢 |
| **集成度 (容量)** | 低 | 高 |
| **成本** | 高 | 低 |
| **常用领域** | 高速缓存 (Cache) | 主存 (内存条) |

## 关键特征

- **[[DRAM刷新]]**: 由于电容上的电荷会随时间流失，必须定期补充电荷。
- **地址复用技术**: DRAM 通常将地址线分为行地址和列地址，分两次输入，以减少芯片引脚数量。 #重点
