ASCII（American Standard Code for Information Interchange）是基于拉丁字母的一套电脑编码系统。

## 特点

- **编码长度**: 使用 7 位二进制位表示，共 128 个字符。
- **存储**: 在计算机中通常占用 1 个字节（8 位），最高位常作为校验位或置 0。 #重点

## 字符分类

1. **控制字符 (0-31, 127)**: 如回车 (CR)、换行 (LF)、退格 (BS) 等。
2. **可打印字符 (32-126)**:
    - **数字 (48-57)**: `0` 的 ASCII 码为 $30H$。
    - **大写字母 (65-90)**: `A` 的 ASCII 码为 $41H$。 #重点
    - **小写字母 (97-122)**: `a` 的 ASCII 码为 $61H$。

## 规律与推导 #重点

- **大小关系**: 数字 < 大写字母 < 小写字母。
- **转换**: 大写字母与对应小写字母的 ASCII 码相差 $32$ ($20H$)。
- **连续性**: 只要记住一个基准值（如 `A` 是 $41H$），就可以推导出其他字母的编码。

作为最基础的 [[非数值数据的编码表示]]，ASCII 码为后续的汉字编码提供了区分基础。
