# 主存读写操作过程

处理器对存储器的访问通过一系列时序信号控制，核心在于地址、数据与控制信号的协同。

## 读操作 (Read)
1. CPU 将地址信息通过 [[MAR与MDR|MAR]] 送往地址总线，由主存接收。
2. CPU 发出“读”命令信号。
3. CPU **等待并监控 [[MFC信号]]**（反馈信号）。
4. 主存根据地址找到单元，将内容送上数据总线。
5. 数据进入 [[MAR与MDR|MDR]]，读操作完成。

## 写操作 (Write) #重点 
1. CPU **同时准备**好地址信息和数据信息：
    - 地址送入 [[MAR与MDR|MAR]]。
    - 待写数据送入 [[MAR与MDR|MDR]]。
2. 地址和数据分别经总线发送给主存，主存先激活并定位对应存储单元。
3. CPU 发出“写”操作指令。
4. 主存接收指令，将总线上的数据写入选中的存储单元。
5. 主存操作结束后反馈 [[MFC信号]]，CPU 接收到信号后结束写周期。
