有权码，四位权值为 `2, 4, 2, 1`。权值总和为 9，确保十进制数 0-9 的唯一编码。

- **自补性**：数字 N 的码与 (9−N) 的码相加为 `1111`（即十进制 9），简化减法为加法运算。
- 适用于计算器与算盘的数字化设计。

| --- | ------ |
| 0   | 0000   |
| 1   | 0001   |
| 2   | 0010   |
| 3   | 0011   |
| 4   | 0100   |
| 5   | 0101   |
| 6   | 0110   |
| 7   | 0111   |
| 8   | 1110   |
| 9   | 1111   |
